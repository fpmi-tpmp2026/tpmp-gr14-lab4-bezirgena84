# Country Management System

## Project Name
Country Management System

## Description
Country Management System - это приложение для управления информацией о странах и их регионах (областях, штатах, провинциях). Система позволяет выполнять CRUD-операции (создание, чтение, обновление, удаление) над данными о странах и регионах. Приложение включает аутентификацию пользователей и хранит данные в реляционной базе данных SQLite. Система предоставляет удобный интерфейс для работы с географическими данными, включая информацию о населении, площади, валюте и административных центрах.

## Installation

### Требования
- Python 3.8 или выше
- SQLite3
- Git

1. Клонируйте репозиторий:
```bash
git clone https://github.com/fpmi-tpmp2026/tpmp-gr14-lab4-bezirgena84.git

```

## Usage

1. **Запуск приложения**: После установки запустите `python main.py`

2. **Авторизация**: 
   - Введите логин и пароль для входа в систему
   - Для первого входа используйте: admin/admin

3. **Основное меню**:
   - Выберите "CRUD Countries" для управления странами
   - Выберите "CRUD Regions" для управления регионами
   - Выберите "Logout" для выхода из системы

4. **Управление странами**:
   - Добавьте новую страну с указанием названия, столицы, языка, населения, площади, валюты и главы государства
   - Просмотрите список всех стран
   - Обновите информацию о стране
   - Удалите страну (все связанные регионы будут удалены автоматически)

5. **Управление регионами**:
   - Добавьте новый регион, указав название, административный центр, население и площадь
   - Привяжите регион к существующей стране
   - Просмотрите, обновите или удалите регионы

# Country Management System

## Project Name
Country Management System

## Description
Country Management System - это приложение для управления информацией о странах и их регионах (областях, штатах, провинциях). Система позволяет выполнять CRUD-операции (создание, чтение, обновление, удаление) над данными о странах и регионах. Приложение включает аутентификацию пользователей и хранит данные в реляционной базе данных SQLite. Система предоставляет удобный интерфейс для работы с географическими данными, включая информацию о населении, площади, валюте и административных центрах.

## Installation

### Требования
- GCC компилятор
- SQLite3
- OpenSSL (для хэширования паролей)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/fpmi-tpmp2026/tpmp-gr14-lab4-bezirgena84.git

```

## Usage

1. **Запуск приложения**:  
```bash
./bin/country_manager
```

2. **Авторизация**: 
   - При первом запуске в системе уже создан администратор:
    Username: admin
    Password: admin

3. **Управление странами**:
   - Добавление страны с указанием названия, столицы, населения и пути к изображению флага
   - Просмотр списка всех стран
   - Обновление информации о стране
   - Удаление страны (все связанные регионы удаляются автоматически)

4. **Управление регионами**:
   - Добавление региона для выбранной страны
   - Просмотр регионов по стране
   - Обновление информации о регионе
   - Удаление региона

5. **Аналитические запросы**:
   - Список регионов указанной страны
   - Среднее население по регионам для указанной страны
   - Общая сумма населения всех стран в базе данных
